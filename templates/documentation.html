1. The structure of the JSON file

The JSON file is used to describe the structure of the dataset. A node table should be made for each table in your dataset. It contains the configuration on how to handle this table. It has the following elements:

metadata.json


  • "dataset name": "example",
  • "license": "CC0" ,
  • "description": "This is an example dataset" ,
  • "number of tables": 3 ,
  • "tables": {
  •   "users": {
  •    "fields": {
  •       "country":{
  •         "type": "string",
  •         "description": "The country of the user"
  •       ...
  •       },
  •     }
  •    "path": "users.csv",
  •    "primary_key": "user_id",
  •   }
  • ...
  • }
For more information on the structure of the JSON file, please refer to the SDV documentation.
